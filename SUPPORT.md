# Support

## 获取帮助

请先按顺序查看：

1. `README.md`
2. `docs/README.md`
3. `docs/code-standard.md`
4. `docs/development-workflow.md`
5. `docs/testing-strategy.md`
6. 如涉及部署或配置，查看 `docs/deploy-runbook.md`

## 提问时请提供

- 目标
- 现象
- 复现步骤
- 错误信息
- 环境信息（Node / pnpm / OS）

## 问题分类

- 功能问题：提 `feature request`
- 缺陷问题：提 `bug report`
- 架构或流程讨论：先引用相关文档或 PR
